# SharkFlow Mobile Client

SharkFlowMobile is a native Android application designed to give developers and project managers a fluid, high-performance, and responsive task tracking experience on the go.

Built strictly in native Kotlin, the application adheres to modern Android development standards, ensuring optimal memory usage and battery efficiency.

## 🛠️ Android Tech Stack

*   **Language**: Native Kotlin.
*   **Architecture**: Modern Jetpack Architecture (MVVM - Model-View-ViewModel) enforcing a strict separation of UI and business logic.
*   **UI Components**: Material Design components, Jetpack Compose / XML layouts, and custom interactive board views.
*   **Async & Concurrency**: Kotlin Coroutines and Flow for non-blocking asynchronous operations and real-time data streaming.
*   **Network & Sync**: Retrofit2 for REST API consumption and secure WebSockets for real-time board state synchronization.

## 🚀 Key Features

*   **Real-Time Board Sync**: Uses WebSockets to receive instant updates when tasks are edited or moved on the React web client, updating the mobile UI with zero delay.
*   **Local Offline Caching**: Implements local database storage to allow users to view their boards and create offline tasks, which automatically synchronize with the backend once internet connection is restored.
*   **Smooth Fluid Layouts**: Responsive task cards, nested checklist adapters, and smooth transitions optimized for varying Android device screen sizes.
